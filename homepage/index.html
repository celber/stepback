I want to  my code.

StepBack is simple and low-level, plain Javascript application framework. Create incredibly fast and beautiful with ease.

If you know Javascript, you already know StepBack!

I find that with modern frameworks current web development has went in wrong direction. We no longer develop applications directly but let them be transpilled or generated by a framework. This indeed introduces faster development and, in some cases, stability, but drastiacally decreases code quality and app performance. This is why i developed StepBack, a modern but plain javascript framework which allows to take back control about low-level application behaviour. I belive every experienced developer knows plain Javascript good enough to create better code than other frameworks generate. It requires very little to include in vanilla Javascript to clean up more advanced scripts without overhead.

Main assumptions

StepBack provides what you love in bigger frameworks, without overhead.

Create class-like components

Easily define constructor which inherits methods and parameters from other components.

Best of ECMA6

We have picked up the best and most commonly used features and included them in StepBack.

DOM isolation

Unlike in jQuery, componenets are created first in memory and rendered only when they are ordered to do so.

Documentation

We aim to acheive easy to read documentation but provide as big coverage as possible.

Using a new framework does not need to feel like learning new language from scratch!

No transpiling or autogeneration

Transpilers change, provide unwanted heavy abstractions and overcomplicate some basic mechanisms. In StepBack you are writing end product directly, you are in charge.
